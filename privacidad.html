Política de Privacidad

Última actualización: Mayo de 2026

1. Recopilación de Datos: La aplicación "Dato Crítico" es una herramienta de uso estrictamente interno. NO recopilamos ni compartimos datos personales de usuarios externos.

2. Uso de la API de TikTok: Utilizamos la API exclusivamente para subir contenido de vídeo a nuestra propia cuenta.

3. Compartir Información: No vendemos ni compartimos ningún tipo de información con entidades externas.
